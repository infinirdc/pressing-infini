# Infini Pressing Website

This project is a one-page responsive website for Infini Pressing, replicating the design and content from provided JPG flyers.

## Setup and Running Locally

To view this project locally, follow these steps:

1.  **Clone the repository (if applicable) or ensure all files are in a single directory.**

2.  **Open `index.html` in your web browser.**

    Simply double-click the `index.html` file, or right-click and choose "Open with" your preferred web browser (e.g., Chrome, Firefox, Edge).

    The website should load directly in your browser.

## Features

*   **Responsive Design:** Adapts to different screen sizes (mobile-first approach).
*   **HERO Section:** Replicates the front page of the flyer with a two-column layout on desktop.
*   **PRIX Section:** Displays the price list from the second flyer page.
*   **Fixed Contact Bar:** A sticky WhatsApp contact bar at the bottom of the screen.
*   **Interactive Elements:** WhatsApp button opens a chat, and service items have hover effects.
*   **Custom Icons:** SVG icons for services, checkmarks, delivery, and WhatsApp.
